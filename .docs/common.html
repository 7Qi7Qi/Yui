判断队列空和满的3种方式

一、少用一个存储位
Ⅰ. 当没有下标，入链表相连

  1. 队满 rear + 1 == front

  2. 队空 rear == front

Ⅱ.当有下标标识

  1. 队满 (rear + 1 ) % maxsize == front

  2. 队空 rear == front

二、设置一个标识位

  1. 设置初始标记 boolean flag = false;

  2. 当入队时，让rear向后移动，flag = true;

  3. 当出队时，让front向后移动，flag = false;（相反亦可，保持同向移动即可）

  4. 判断

    1. 队满 rear == front && flag == true

    2. 队空 rear == front && flag == false

三、计数count,队列中有效元素的个数

  1. 队满 count == maxsize

  2. 队空 count == 0
